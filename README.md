# Autonomous Drone Navigation

This project implements a vision-based autonomous drone navigation system using:

- ROS
- OpenCV
- YOLO
- Gazebo simulation

The system detects obstacles and automatically adjusts drone movement.

## Features

- Real-time obstacle detection
- Autonomous navigation
- Simulation environment
- Computer vision pipeline

## Tech Stack

Python  
OpenCV  
YOLOv8  
ROS  
Gazebo
